The admission method tells you how the the patient arrived into the hospital

Note: The release of version he release of version 6.2 of the Commissioning Data Sets (CDS) introduced an updated set of admission method codes to be used from 1st April 2013 onwards. Admission method code 25 was introduced as part of this update and admission method code 28 was replaced with codes 2A, 2B, 2C, 2D. Historic data with admission method code 28 will continue to be categorised as ‘Acute’ admissions.
